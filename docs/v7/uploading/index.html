Uploading

This feature is part of the Pro Player plan. Pricing information can be found here.

Upload user-generated video content from your React Native app. Supports large file uploads with resumable transfers, compression, and background uploading.

Featuresâ€‹

  • Resumable uploads - Continue interrupted uploads from where they left off
  • Background uploads - Keep uploading when app is backgrounded
  • Progress tracking - Real-time upload progress and speed
  • Video compression - Reduce file size before upload
  • Chunked uploads - Split large files for reliable transfer
  • Multiple providers - Support for S3, GCS, Azure, and custom servers

Basic Usageâ€‹

Upload a Videoâ€‹

import { uploadVideo } from 'react-native-video-upload';

async function handleUpload(videoUri: string) {
  const upload = await uploadVideo({
    source: videoUri,
    destination: 'https://api.example.com/upload',
    // Optional metadata
    metadata: {
      title: 'My Video',
      description: 'A great video',
    },
  });
  
  console.log('Upload URL:', upload.url);
}

Track Progressâ€‹

import { uploadVideo, useUploadProgress } from 'react-native-video-upload';

function UploadButton({ videoUri }) {
  const [uploadId, setUploadId] = useState(null);
  const { progress, speed, timeRemaining } = useUploadProgress(uploadId);
  
  const startUpload = async () => {
    const upload = await uploadVideo({
      source: videoUri,
      destination: 'https://api.example.com/upload',
      onProgress: (p) => console.log(`${p}%`),
    });
    setUploadId(upload.id);
  };
  
  return (
    <View>
      <Button title="Upload" onPress={startUpload} />
      {uploadId && (
        <>
          <ProgressBar progress={progress} />
          <Text>Speed: {speed} MB/s</Text>
          <Text>Time remaining: {timeRemaining}s</Text>
        </>
      )}
    </View>
  );
}

Video Compressionâ€‹

Reduce file size before uploading:

import { compressVideo, uploadVideo } from 'react-native-video-upload';

async function compressAndUpload(videoUri: string) {
  // Compress the video first
  const compressed = await compressVideo(videoUri, {
    quality: 'medium', // 'low' | 'medium' | 'high' | 'original'
    maxWidth: 1920,
    maxHeight: 1080,
    bitrate: 5000000, // 5 Mbps
  });
  
  console.log(`Original: ${compressed.originalSize} bytes`);
  console.log(`Compressed: ${compressed.compressedSize} bytes`);
  console.log(`Saved: ${compressed.savedPercentage}%`);
  
  // Then upload
  const upload = await uploadVideo({
    source: compressed.uri,
    destination: 'https://api.example.com/upload',
  });
  
  return upload;
}

Resumable Uploadsâ€‹

Handle interrupted uploads:

import { uploadVideo, resumeUpload, getPendingUploads } from 'react-native-video-upload';

// Start upload with resumable option
const upload = await uploadVideo({
  source: videoUri,
  destination: 'https://api.example.com/upload',
  resumable: true,
});

// Later, if interrupted, get pending uploads
const pending = await getPendingUploads();

// Resume each pending upload
for (const upload of pending) {
  await resumeUpload(upload.id);
}

Background Uploadsâ€‹

Continue uploading when app is backgrounded:

import { configureBackgroundUploads, uploadVideo } from 'react-native-video-upload';

// Configure background behavior (call once at app start)
configureBackgroundUploads({
  // iOS
  sessionIdentifier: 'com.myapp.upload',
  // Android
  showNotification: true,
  notificationTitle: 'Uploading video...',
});

// Upload will continue in background
const upload = await uploadVideo({
  source: videoUri,
  destination: 'https://api.example.com/upload',
  backgroundUpload: true,
});

Eventsâ€‹

Listen to upload lifecycle events:

import { addUploadListener } from 'react-native-video-upload';

useEffect(() => {
  const subscription = addUploadListener((event) => {
    switch (event.type) {
      case 'started':
        console.log('Upload started');
        break;
      case 'progress':
        console.log(`Progress: ${event.progress}%`);
        break;
      case 'completed':
        console.log('Upload URL:', event.url);
        break;
      case 'error':
        console.error('Upload failed:', event.error);
        break;
      case 'cancelled':
        console.log('Upload cancelled');
        break;
    }
  });
  
  return () => subscription.remove();
}, []);
