Features‚Äã

  • Device discovery - Automatic detection of nearby cast devices
  • Seamless handoff - Continue playback when casting starts/stops
  • Remote control - Control playback from the phone while casting
  • Queue management - Build playlists for continuous playback

Cast Button‚Äã

Add a cast button to your player controls:

import Video from 'react-native-video';
import { CastButton } from 'react-native-video-pro';

function PlayerWithCast() {
  return (
    <View style={styles.container}>
      <Video
        source={{ uri: 'https://example.com/video.mp4' }}
        style={styles.video}
      />
      <View style={styles.controls}>
        <CastButton 
          style={styles.castButton}
          tintColor="#96c9eb"
        />
      </View>
    </View>
  );
}

Chromecast Integration‚Äã

import { useChromecast } from 'react-native-video-pro';

function ChromecastPlayer() {
  const { 
    isConnected, 
    deviceName, 
    connect, 
    disconnect,
    castMedia 
  } = useChromecast();

  const startCasting = async () => {
    if (!isConnected) {
      await connect();
    }
    
    await castMedia({
      source: 'https://example.com/video.mp4',
      title: 'My Video',
      subtitle: 'Video description',
      imageUrl: 'https://example.com/thumbnail.jpg',
    });
  };

  return (
    <View>
      {isConnected ? (
        <Text>Connected to {deviceName}</Text>
      ) : (
        <Button title="Cast" onPress={startCasting} />
      )}
    </View>
  );
}

Airplay Integration‚Äã

import { useAirplay } from 'react-native-video-pro';

function AirplayPlayer() {
  const { 
    isAvailable, 
    isActive,
    showPicker 
  } = useAirplay();

  if (!isAvailable) {
    return null;
  }

  return (
    <TouchableOpacity onPress={showPicker}>
      <AirplayIcon active={isActive} />
    </TouchableOpacity>
  );
}

Events‚Äã

Handle cast state changes:

<Video
  source={{ uri: 'https://example.com/video.mp4' }}
  onCastStateChanged={(state) => {
    switch (state) {
      case 'connecting':
        // Show loading indicator
        break;
      case 'connected':
        // Hide local controls
        break;
      case 'disconnected':
        // Resume local playback
        break;
    }
  }}
  onCastError={(error) => {
    console.error('Cast error:', error);
  }}
/>
