Chapters

This feature is part of the Pro Player plan. Pricing information can be found here.

Video chapters allow users to navigate directly to specific sections of your video, improving user experience for long-form content like tutorials, courses, or podcasts.

Featuresâ€‹

  • Chapter markers - Visual indicators on the seekbar
  • Chapter list - Sidebar or overlay with chapter titles
  • Thumbnails - Preview images for each chapter
  • Auto-detection - Parse chapters from video metadata
  • Custom chapters - Define chapters programmatically

Basic Usageâ€‹

import Video from 'react-native-video';
import { ChapterList } from 'react-native-video-pro';

const chapters = [
  { id: '1', title: 'Introduction', startTime: 0, thumbnail: 'intro.jpg' },
  { id: '2', title: 'Getting Started', startTime: 120, thumbnail: 'start.jpg' },
  { id: '3', title: 'Advanced Topics', startTime: 480, thumbnail: 'advanced.jpg' },
  { id: '4', title: 'Conclusion', startTime: 900, thumbnail: 'end.jpg' },
];

function VideoWithChapters() {
  const videoRef = useRef(null);

  const onChapterSelect = (chapter) => {
    videoRef.current?.seek(chapter.startTime);
  };

  return (
    <View style={styles.container}>
      <Video
        ref={videoRef}
        source={{ uri: 'https://example.com/video.mp4' }}
        chapters={chapters}
        style={styles.video}
      />
      <ChapterList 
        chapters={chapters}
        onChapterSelect={onChapterSelect}
      />
    </View>
  );
}

Chapter Markers on Seekbarâ€‹

Display chapter markers directly on the progress bar:

import { SeekbarWithChapters } from 'react-native-video-pro';

<SeekbarWithChapters
  chapters={chapters}
  currentTime={currentTime}
  duration={duration}
  onSeek={handleSeek}
  markerColor="#96c9eb"
/>

Auto-detect from Metadataâ€‹

Some video formats include chapter information in metadata:

<Video
  source={{ uri: 'https://example.com/video-with-chapters.mp4' }}
  onChaptersLoaded={(chapters) => {
    console.log('Detected chapters:', chapters);
  }}
/>
