Custom Subtitles

This feature is part of the Pro Player plan. Pricing information can be found here.

Take full control over subtitle rendering with customizable fonts, colors, positioning, and effects. Create a unique viewing experience that matches your brand.

Featuresâ€‹

  • Custom styling - Fonts, colors, backgrounds, shadows
  • Positioning - Place subtitles anywhere on screen
  • Animations - Fade, slide, and custom transitions
  • Multiple tracks - Switch between subtitle languages
  • Accessibility - Full VoiceOver/TalkBack support

Basic Usageâ€‹

import Video from 'react-native-video';
import { SubtitleOverlay } from 'react-native-video-pro';

function PlayerWithSubtitles() {
  const [currentSubtitle, setCurrentSubtitle] = useState(null);

  return (
    <View style={styles.container}>
      <Video
        source={{ uri: 'https://example.com/video.mp4' }}
        textTracks={[
          {
            title: 'English',
            language: 'en',
            type: 'text/vtt',
            uri: 'https://example.com/subtitles-en.vtt',
          },
          {
            title: 'Spanish',
            language: 'es',
            type: 'text/vtt',
            uri: 'https://example.com/subtitles-es.vtt',
          },
        ]}
        selectedTextTrack={{ type: 'language', value: 'en' }}
        onTextTrackDataChanged={(data) => setCurrentSubtitle(data)}
        style={styles.video}
      />
      
      <SubtitleOverlay
        text={currentSubtitle?.text}
        style={styles.subtitle}
      />
    </View>
  );
}

Styling Optionsâ€‹

Customize every aspect of subtitle appearance:

<SubtitleOverlay
  text={currentSubtitle?.text}
  style={{
    // Container styling
    position: 'absolute',
    bottom: 60,
    left: 20,
    right: 20,
    alignItems: 'center',
  }}
  textStyle={{
    // Text styling
    fontFamily: 'Inter',
    fontSize: 18,
    fontWeight: '600',
    color: '#ffffff',
    textShadowColor: 'rgba(0,0,0,0.8)',
    textShadowOffset: { width: 1, height: 1 },
    textShadowRadius: 3,
  }}
  backgroundStyle={{
    // Background styling
    backgroundColor: 'rgba(0,0,0,0.6)',
    paddingHorizontal: 16,
    paddingVertical: 8,
    borderRadius: 8,
  }}
/>

Animated Subtitlesâ€‹

Add transitions between subtitle changes:

import { AnimatedSubtitle } from 'react-native-video-pro';

<AnimatedSubtitle
  text={currentSubtitle?.text}
  animation="fade" // 'fade' | 'slide' | 'scale' | 'custom'
  duration={200}
/>

Subtitle Track Selectorâ€‹

Let users choose their preferred language:

import { SubtitleSelector } from 'react-native-video-pro';

<SubtitleSelector
  tracks={textTracks}
  selectedTrack={selectedTrack}
  onSelect={(track) => setSelectedTrack(track)}
  offLabel="Off"
  style={styles.selector}
/>

Karaoke-style Highlightingâ€‹

Highlight words as they're spoken:

<SubtitleOverlay
  text={currentSubtitle?.text}
  wordTimings={currentSubtitle?.wordTimings}
  currentTime={currentTime}
  highlightColor="#96c9eb"
/>

Accessibilityâ€‹

Ensure subtitles are accessible:

<SubtitleOverlay
  text={currentSubtitle?.text}
  accessible={true}
  accessibilityLabel={`Subtitle: ${currentSubtitle?.text}`}
  accessibilityRole="text"
  // Respect system font size
  allowFontScaling={true}
/>
