Basic Controls

Build your own video player UI using React Native Video's compound component API. Use ready-made components and arrange them however you want.

Overviewâ€‹

The compound API provides ready-to-use components that handle all the video logic. You just arrange them:

  • PlayButton - Play button component
  • PauseButton - Pause button component
  • Seekbar - Progress bar with scrubbing
  • VolumeControl - Volume slider and mute button
  • FullscreenButton - Fullscreen toggle
  • TimeDisplay - Current time and duration
  • LoadingIndicator - Buffering state

VideoProviderâ€‹

All components must be wrapped in a VideoProvider that manages the video state:

import { VideoProvider } from 'react-native-video';

function App() {
  return (
    <VideoProvider
      source={{ uri: 'https://example.com/video.mp4' }}
    >
      {/* Your custom UI components here */}
    </VideoProvider>
  );
}

Basic Exampleâ€‹

Arrange components however you want:

import React from 'react';
import { View, StyleSheet } from 'react-native';
import {
  VideoProvider,
  PlayButton,
  PauseButton,
  Seekbar,
  TimeDisplay,
} from 'react-native-video';

function CustomPlayer() {
  return (
    <VideoProvider
      source={{ uri: 'https://example.com/video.mp4' }}
      style={styles.video}
    >
      <View style={styles.container}>
        {/* Video renders automatically */}
        
        <View style={styles.controls}>
          <PlayButton style={styles.button} />
          <PauseButton style={styles.button} />
          
          <Seekbar style={styles.seekbar} />
          
          <TimeDisplay style={styles.time} />
        </View>
      </View>
    </VideoProvider>
  );
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
  },
  video: {
    width: '100%',
    height: 300,
  },
  controls: {
    flexDirection: 'row',
    alignItems: 'center',
    padding: 16,
    backgroundColor: 'rgba(0,0,0,0.5)',
  },
  button: {
    padding: 8,
  },
  seekbar: {
    flex: 1,
    marginHorizontal: 16,
  },
  time: {
    color: 'white',
    fontSize: 12,
  },
});

Component Referenceâ€‹

PlayButtonâ€‹

Renders a play button that automatically shows when video is paused:

import { PlayButton } from 'react-native-video';

<PlayButton 
  style={styles.button}
  iconStyle={styles.icon}
/>

PauseButtonâ€‹

Renders a pause button that automatically shows when video is playing:

import { PauseButton } from 'react-native-video';

<PauseButton 
  style={styles.button}
  iconStyle={styles.icon}
/>

Seekbarâ€‹

Interactive progress bar with scrubbing:

import { Seekbar } from 'react-native-video';

<Seekbar 
  style={styles.seekbar}
  trackColor="rgba(255,255,255,0.3)"
  progressColor="#96c9eb"
  thumbColor="#96c9eb"
/>

VolumeControlâ€‹

Volume slider with mute button:

import { VolumeControl } from 'react-native-video';

<VolumeControl 
  style={styles.volume}
  showMuteButton={true}
  sliderStyle={styles.slider}
/>

FullscreenButtonâ€‹

Toggle fullscreen mode:

import { FullscreenButton } from 'react-native-video';

<FullscreenButton 
  style={styles.button}
  iconStyle={styles.icon}
/>

TimeDisplayâ€‹

Shows current time and duration:

import { TimeDisplay } from 'react-native-video';

<TimeDisplay 
  style={styles.time}
  format="mm:ss" // or "hh:mm:ss"
  showDuration={true}
/>

LoadingIndicatorâ€‹

Shows when video is buffering:

import { LoadingIndicator } from 'react-native-video';

<LoadingIndicator 
  style={styles.loader}
  color="#96c9eb"
/>

Customizationâ€‹

All components accept style prop and can be customized:

<PlayButton 
  style={{
    backgroundColor: '#96c9eb',
    borderRadius: 20,
    padding: 12,
  }}
  iconStyle={{
    color: '#171717',
    size: 24,
  }}
/>

<Seekbar
  style={{
    height: 6,
    marginVertical: 16,
  }}
  trackColor="rgba(255,255,255,0.2)"
  progressColor="#f4d970"
  thumbColor="#f4d970"
  thumbSize={16}
/>

Complete Exampleâ€‹

import React from 'react';
import { View, StyleSheet } from 'react-native';
import {
  VideoProvider,
  PlayButton,
  PauseButton,
  Seekbar,
  TimeDisplay,
  VolumeControl,
  FullscreenButton,
  LoadingIndicator,
} from 'react-native-video';

function CompletePlayer() {
  return (
    <VideoProvider
      source={{ uri: 'https://example.com/video.mp4' }}
      style={styles.video}
    >
      <View style={styles.container}>
        <LoadingIndicator style={styles.loader} />
        
        <View style={styles.controls}>
          <View style={styles.topRow}>
            <PlayButton style={styles.button} />
            <PauseButton style={styles.button} />
            <Seekbar style={styles.seekbar} />
            <TimeDisplay style={styles.time} />
          </View>
          
          <View style={styles.bottomRow}>
            <VolumeControl style={styles.volume} />
            <FullscreenButton style={styles.button} />
          </View>
        </View>
      </View>
    </VideoProvider>
  );
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
  },
  video: {
    width: '100%',
    height: 300,
  },
  loader: {
    position: 'absolute',
    top: '50%',
    left: '50%',
    transform: [{ translateX: -20 }, { translateY: -20 }],
  },
  controls: {
    position: 'absolute',
    bottom: 0,
    left: 0,
    right: 0,
    padding: 16,
    backgroundColor: 'rgba(0,0,0,0.7)',
  },
  topRow: {
    flexDirection: 'row',
    alignItems: 'center',
    marginBottom: 8,
  },
  bottomRow: {
    flexDirection: 'row',
    alignItems: 'center',
    justifyContent: 'space-between',
  },
  button: {
    padding: 8,
  },
  seekbar: {
    flex: 1,
    marginHorizontal: 16,
  },
  time: {
    color: 'white',
    fontSize: 12,
    minWidth: 80,
  },
  volume: {
    flexDirection: 'row',
    alignItems: 'center',
  },
});
