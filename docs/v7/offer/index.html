Offer

react-native-video is community-first and stays free to use.

Pro Player is an optional set of add-ons and maintainer services for teams that need advanced features or faster support.

Contact usI have an urgent issue

What you can use​

Three ways to work with react-native-video, depending on your needs and timeline:

Community

Access comprehensive documentation, browse code examples, and get help through GitHub issues and community discussions. Support is best-effort based on maintainer availability and community engagement.

Get started→

Maintainer support

Get priority assistance directly from the core maintainers when you need faster response times than community support can provide. Available as one-time fixes or ongoing partnerships.

Issue BoosterSupport plan
See services→

Extensions

Enhance your video implementation with optional building blocks: offline playback capabilities, chapters navigation, custom UI components, and production-ready starter templates.

Offline VideoChapters / Custom UIBoilerplates
See extensions→

How it works​

Community Support→Search docs & examples→Post issue / discussion→Wait for community response
Issue Booster→Submit bug report with repro→Maintainer prioritizes & fixes→Receive patch/PR/release
Support Plan→Define your needs & scope→Get ongoing maintainer access→Ship with confidence

Most users only need the community path - the other options exist for teams with tighter deadlines or advanced requirements.

Which option should I choose?​

Quick reference guide based on your situation:

SituationChooseResult
"I'm just integrating video"CommunityBest-effort help + docs
"A crash/bug blocks release"Issue BoosterASAP fix (PR/patch/release)
"We need ongoing maintainer help"Support planStability + guidance over time
"We want to switch from another player"MigrationLower costs + full control
"We need offline / chapters / custom UI"ExtensionsAdvanced capabilities
