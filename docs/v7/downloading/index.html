Downloading

This feature is part of the Pro Player plan. Pricing information can be found here.

Enable offline video playback in your app. Download HLS streams with full support for DRM, multiple audio tracks, and subtitles.

Featuresâ€‹

  • HLS downloading - Download and store HLS content for offline playback
  • Offline DRM - Secure playback of protected content offline
  • Multiple audio tracks - Choose which languages to download
  • Subtitles - Include subtitle tracks in downloads
  • Background downloads - Continue downloading when app is backgrounded
  • Progress tracking - Monitor download progress and status
  • Storage management - Query and manage downloaded content

Installationâ€‹

npm install react-native-video-offline

Basic Usageâ€‹

Download a Videoâ€‹

import { downloadVideo, getDownloads } from 'react-native-video-offline';

async function startDownload() {
  const download = await downloadVideo({
    id: 'video-123',
    source: 'https://example.com/video.m3u8',
    title: 'My Video',
    // Optional: specify which tracks to download
    audioTracks: ['en', 'es'],
    subtitleTracks: ['en'],
  });
  
  console.log('Download started:', download.id);
}

Monitor Progressâ€‹

import { useDownloadProgress } from 'react-native-video-offline';

function DownloadItem({ videoId }) {
  const { progress, status, bytesDownloaded, totalBytes } = useDownloadProgress(videoId);
  
  return (
    <View>
      <Text>{status}</Text>
      <ProgressBar progress={progress} />
      <Text>{bytesDownloaded} / {totalBytes}</Text>
    </View>
  );
}

Play Offline Contentâ€‹

import Video from 'react-native-video';
import { getOfflineSource } from 'react-native-video-offline';

function OfflinePlayer({ videoId }) {
  const [source, setSource] = useState(null);
  
  useEffect(() => {
    async function loadSource() {
      const offlineSource = await getOfflineSource(videoId);
      setSource(offlineSource);
    }
    loadSource();
  }, [videoId]);
  
  if (!source) return <Text>Loading...</Text>;
  
  return (
    <Video
      source={source}
      style={{ width: '100%', height: 300 }}
    />
  );
}

Download Managementâ€‹

Pause/Resume Downloadsâ€‹

import { pauseDownload, resumeDownload, cancelDownload } from 'react-native-video-offline';

// Pause
await pauseDownload('video-123');

// Resume
await resumeDownload('video-123');

// Cancel
await cancelDownload('video-123');

List Downloadsâ€‹

import { getDownloads } from 'react-native-video-offline';

const downloads = await getDownloads();

downloads.forEach(download => {
  console.log(`${download.title}: ${download.status} (${download.progress}%)`);
});

Delete Downloaded Contentâ€‹

import { deleteDownload } from 'react-native-video-offline';

await deleteDownload('video-123');

Check Storageâ€‹

import { getStorageInfo } from 'react-native-video-offline';

const info = await getStorageInfo();
console.log(`Used: ${info.usedBytes}`);
console.log(`Available: ${info.availableBytes}`);

DRM Supportâ€‹

Download protected content with persistent licenses:

const download = await downloadVideo({
  id: 'protected-video',
  source: 'https://example.com/protected.m3u8',
  drm: {
    type: 'widevine', // or 'fairplay'
    licenseServer: 'https://license.example.com',
    headers: {
      'Authorization': 'Bearer token',
    },
    // Request persistent license for offline playback
    persistentLicense: true,
  },
});

Background Downloadsâ€‹

Downloads continue when the app is backgrounded:

import { configureBackgroundDownloads } from 'react-native-video-offline';

// Configure background behavior
configureBackgroundDownloads({
  // iOS: Allow downloads on cellular
  allowsCellularAccess: false,
  // iOS: Continue in background
  isDiscretionary: false,
  // Android: Show notification
  showNotification: true,
  notificationTitle: 'Downloading video...',
});

Eventsâ€‹

Listen to download events:

import { addDownloadListener } from 'react-native-video-offline';

useEffect(() => {
  const subscription = addDownloadListener((event) => {
    switch (event.type) {
      case 'progress':
        console.log(`Progress: ${event.progress}%`);
        break;
      case 'completed':
        console.log('Download completed!');
        break;
      case 'error':
        console.error('Download failed:', event.error);
        break;
    }
  });
  
  return () => subscription.remove();
}, []);
